# Distributed Systems Playground

A collection of Kubernetes manifests and Go application examples for learning distributed systems.

## Applications
- **Go Standard App**: A basic Go application demonstrating standard library usage.

## Workloads
- **ReplicaSet**: Ensuring a specified number of pod replicas are running.
- **DaemonSet**: Running a copy of a pod on all (or some) nodes.

## Batch Processing
- **Jobs**: Run to completion tasks.
- **CronJobs**: Time-based job scheduling.

## Networking & Storage
- **Services**: Exposing applications.
- **Ingress**: HTTP/HTTPS routing.
- **Storage**: Persistent Volumes and Claims.
